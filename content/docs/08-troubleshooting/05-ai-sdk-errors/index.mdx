# AI SDK Errors

The Vercel AI SDK uses `AISDKError` to represent errors that occur during the execution of the SDK.

The following are the different types of errors that can be thrown by the SDK:

- [ APICallError ](/docs/troubleshooting/ai-sdk-errors/api-call-error)
- [ EmptyResponseBodyError ](/docs/troubleshooting/ai-sdk-errors/empty-response-body-error)
- [ InvalidPromptError ](/docs/troubleshooting/ai-sdk-errors/invalid-prompt-error)
- [ InvalidResponseDataError ](/docs/troubleshooting/ai-sdk-errors/invalid-response-data-error)
- [ InvalidDataContentError ](/docs/troubleshooting/ai-sdk-errors/ai-invalid-data-content-error)
- [ JSONParseError ](/docs/troubleshooting/ai-sdk-errors/json-parse-error)
- [ LoadAPIKeyError ](/docs/troubleshooting/ai-sdk-errors/load-api-key-error)
- [ TypeValidationError ](/docs/troubleshooting/ai-sdk-errors/type-validation-error)
- [ UnsupportedFunctionalityError ](/docs/troubleshooting/ai-sdk-errors/unsupported-functionality-error)
- [ InvalidArgumentError ](/docs/troubleshooting/ai-sdk-errors/invalid-argument-error)
- [ InvalidToolArgumentsError ](/docs/troubleshooting/ai-sdk-errors/invalid-tool-arguments-error)
- [ NoSuchToolError ](/docs/troubleshooting/ai-sdk-errors/no-such-tool-error)
- [ NoObjectGeneratedError ](/docs/troubleshooting/ai-sdk-errors/no-object-generated-error)
- [ InvalidMessageRoleError ](/docs/troubleshooting/ai-sdk-errors/invalid-message-role-error)
- [ DownloadError ](/docs/troubleshooting/ai-sdk-errors/download-error)
- [ RetryError ](/docs/troubleshooting/ai-sdk-errors/retry-error)
- [ MessageConversionError ](/docs/troubleshooting/ai-sdk-errors/message-conversion-error)
